Intent: Add x402 payment gateway to expose 5 Strale capabilities in the 402 ecosystem (402index.io)

## What was built

**New file:** `apps/api/src/routes/x402-gateway.ts`
- Hono route group mounted at `/x402/`
- 5 endpoints: iban-validate, vat-format-validate, paid-api-preflight, ssl-check, sanctions-check
- No `X-PAYMENT` header → returns HTTP 402 with `Payment-Required` header (base64 x402 JSON)
- `X-PAYMENT` header present → executes capability via `getExecutor()`, returns JSON output
- Permissive CORS (`*`) for all x402 routes

**app.ts changes:**
- Import added for x402Route
- `app.use("/x402/*", publicCors)` before the auth middleware
- `app.route("/x402", x402Route)` at the bottom

## Phase 1 vs Phase 2
- **Phase 1 (now):** Any X-PAYMENT header grants access — goal is 402 Index directory listing
- **Phase 2 (future):** Add `@x402/hono` middleware with real Base wallet + facilitator verification

## Env var needed
- `X402_WALLET_ADDRESS` — real Base mainnet wallet address for `payTo` field
- Placeholder: `0x0000000000000000000000000000000000000001`
- `API_BASE_URL` — defaults to `https://api.strale.io`
