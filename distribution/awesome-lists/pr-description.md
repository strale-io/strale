## Add Strale MCP Server

**Strale** is the trust layer for AI agents — 250+ independently tested and scored business data capabilities across 27 countries, accessible via MCP.

### What it does
- IBAN, VAT, LEI, BIC validation
- Company data across EU, Nordic, UK, US, APAC (27 countries)
- Sanctions screening and adverse media checks
- SSL certificate and DNS checks
- EU compliance automation (KYB, AML, GDPR)
- Invoice, receipt, and contract data extraction
- Developer tools (CVE lookup, dependency audit, code review)

### Why it belongs here
- Production MCP server at https://api.strale.io/mcp (Streamable HTTP)
- 250+ tools registered via MCP
- Published npm package: [strale-mcp](https://www.npmjs.com/package/strale-mcp)
- Free tier available (iban-validate, strale_search work without API key)
- Every capability independently tested with trust scoring (SQS)
