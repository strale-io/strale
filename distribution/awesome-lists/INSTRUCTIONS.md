# Submitting Strale to Awesome MCP Server Lists

## Process

For each repo below:

1. **Fork** the repo on GitHub
2. **Clone** your fork locally
3. **Add** the Strale entry in the correct category (see entries.md for exact text)
4. **Commit** with message: `Add Strale MCP Server`
5. **Push** to your fork
6. **Submit PR** using the description in pr-description.md
7. **Wait** — review time is typically 1-7 days per repo

## Tips

- Submit to the largest repos first (punkpeye, wong2) — they have the most visibility
- Space submissions out over 2-3 days to avoid looking spammy
- If a repo requires specific metadata (YAML frontmatter, etc.), adapt the entry
- Check each repo's CONTRIBUTING.md before submitting
- Some repos auto-merge, others require manual review
